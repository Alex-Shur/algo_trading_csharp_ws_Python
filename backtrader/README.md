# Тестирование торговой стратегии на двух скользящих средних

**Источник:** [Backtrader](https://github.com/mementum/backtrader)  
**Стратегия:** main.py

## Параметры тестирования

- **Python версия:** 3.13.9
- **Инструмент:** SBER (Сбербанк)
- **Период данных:** 01.01.2024 - 10.10.2025
- **Объем торговли:** 1 контракт
- **SMA1:** 15 периодов
- **SMA2:** 91 период
- **Тип анализа:** Один полный проход по историческим данным
- **Время выполнения:** 49.7 сек

## Требования и установка

- Python 3.13.9
- Менеджер зависимостей `uv`

### Установка пакета `uv`

```bash
pip install uv
```

### Запуск теста

```bash
uv run main.py
```
