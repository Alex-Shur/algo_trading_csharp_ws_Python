# Тестирование торговой стратегии на двух скользящих средних

**Источник:** [Backtesting](https://github.com/kernc/backtesting.py)  
**Стратегия:** bt_sma.py

## Параметры тестирования

- **Python 3.13.9**
- **Инструмент:** SBER (Сбербанк)
- **Период данных:** 01.01.2024 - 10.10.2025
- **Объем торговли:** 1 контракт
- **SMA1:** 15 периодов
- **SMA2:** 91 период
- **Тип анализа:** Один полный проход по историческим данным
- **Время выполнения:** 3.47 сек

## Требования

- Python 3.13.9
- Пакет `uv` для управления зависимостями

## Установка и запуск

### 1. Установка `uv`

```bash
pip install uv
```

### 2. Запуск теста

```bash
uv run bt_sma.py
```
